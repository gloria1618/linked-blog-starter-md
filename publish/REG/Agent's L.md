to 3rd parties for Contract Duties
- Disclosed Principal (Agent not L if Authorized)
	- if agent discloses existence/id of principal, 3rd party cannot hold authorized agent liable on contract
	- every person repping themself as agent impliedly warrants that they have authority they purport to have 
	- if agent doesnt have such authority, 3rd party can hold agent liable for any damages caused based on breach of this implied warranty
- Unidentified n Undisclosed Principal (Agent Liable)
	- agent liable on contract w 3rd party if:
		- principals id not disclosed to 3rd party (unid), or
		- neither existence/id of principal disclosed (undisclosed)
			- no apparent authority w an undisclosed principal bc no holding out of agent by principal
			- principal cannot ratify contracts
	- 3rd party's election
		- can hold either principal or agent (but not both) liable if principal was undisclosed/identified
		- must elect whom to hold liable
	- No effect on Actual Authority
		- undisclosed principal has no effect on agent's actual authority since actual authority arises from comms bw principal n agent
