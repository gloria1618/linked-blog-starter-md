- in addition to duties expressed in agency agrmnt (written/oral), principal owes agent following:
	- implied by law
		- compensation
			- unless agent agreed to act gratuitously, principal has implied duty to give reasonable compensation
		- reimbursement/indemnification (agent got sued)
			- duty to reimburse for all exp incurred in carrying out agency

Remedies of Agent (when principal breaches duties)
- agent can bring an action against principal for any damages caused
- agent has duty to mitigate damages (eg if principal is not paying agent, agent can discontinue to work and sue)
	- agent can stop incurring additional damages by discont pmts on behalf of principal
