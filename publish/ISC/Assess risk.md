[[SAR]]

Security Assessmt Engagements n Reporting
- incl 
	- performing risk assessment
	- testing ctrls for data on curretn state regarding info sec capabilities
- eng often result in issuance of SAR Sec Assessment Report mgmt
	- NIST Special Publications 800-39, 800-53A Rev 5 has fundamentals on how to perform, key elements
- 1st step in sec assessment eng is defining proc, which relates to def obj n id obj/methods
- Assessment methods incl following
	- Examination
		- anal, obsv, review 1+ assessment obj (job roles, sec specs, sec activ, op activ)
	- Interviewing
		- indiv/group discussions to understand, collect, eval evidence
	- Testing
		- test assessment obj reflecting how obj performs in current v target/expected state
- org must support assessment methods by id
	- tools, tech, methods needed to assess risk
	- risk assessment assumptions
	- constraints
	- roles, resp
	- assessment data collection, processing methods
	- comms methods
	- how assessments conducted
	- assessment freq
	- sources, methods of collecting data on threats
