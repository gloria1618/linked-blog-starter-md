Operating Sys (OS)
- orchestrates global functioning of group of apps, hardware, performance as intermediary bw resources 
- defines parameters for managing sys memory, processes, records, devices, user interfact
- optimize functions, balances resources, allocates to allow sys to run seamlessly w/o delay/interruptions even w mult apps running simultaneously 

Firmware
- software locally embedded in hardware instructs hardware how to op
- op like software but exists locally on machine direction func of phys components, like motherboard n microprocessor 
- not updated freq/at all, very diff from typical software

Mobile Tech

- connectivity typically done w wireless tech like wifi, bluetooth, 4/5G cellular tech
- combines hardware (laptop, tablet, hot spot, mobile phone) w mobile apps n OS that allow connectivity to networks
- allows user to conduct business, comms real time
