[[CIS Control 1-18]]

Center for Internet Security (CIS) Controls 
- rec set of actions, processes, best practices to be adopt/implemented by org to strengthen cybersecurity defenses
	- first dev in '08 by an international consortium, has evolved
	- controls currently supported by SANS Institute (SysAdmin, Audit, Network, Sec) which provides training, administers certification, performs research
	- each control update involves experts (SME) across industries, entity type (gov v public), and job roles (operators, policy makers)

Controls
- task focused, organized by activities
- 18 CIS controls reflecting shifts in cybersec ecosys 
- Safeguards - each has rec prescribed to achieve control objective

Design Principles
- Context
	- enhancement to scope/practical applicability of safeguards thru incorp of ex/explanations
- Coexistence
	- alignment w evolving industry standard/framework, incl NIST CSF 2.0 framework
- Consistency
	- disruption to controls users are minimized, limiting impact on implementation groups

Implementation Groups
- can be tailored to org's size
- self assessed categories that identify subset of CIS Controls critical for org to adopt based on size
- IG1
	- small/med size org w limited cybersec defense mechanism in terms of personnel/IT A
	- focus - keep company operational bc
		- cybersec expertise limited
		- data used not sensitive (no PII PHI)
		- cannot sustain long periods of downtime
	- sim to tier 1/2 partial/riskinformed of NIST Cybersec framework
- IG2 (incl IG1)
	- org has IT staff supporting mult dep w various risk profiles
	- typically have sensitive client data, can tolerate short service interruptions
	- concern - loss of trust if data breach
	- sim to tier 3 repeatable of NIST Cybersec framework
- IG3 (incl IG1/2)
	- org have sec experts in all domains w/in cybersec like penetration testing, risk mgmt, application sec
	- data A under mgmt incl those sensitive and subj to compliance w standards/regulatory oversight
	- attacks can cause sig damage to company/public welfare
	- sim to tier 4/adaptive NIST Cybersec framework
