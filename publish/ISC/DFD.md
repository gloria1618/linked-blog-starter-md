Data Flow Diagram
- describes data flow thru a process

- Process
	-  rounded rectangle or circle
	- action resulting in data changing, producing new output
	- labels sim to BPMN
- Data Flow
	- curved/straight arrow
	- direction of data flow bw processes
	- ex: customer contact deets
- Data Store/Warehouse
	- open ended rectangle
	- where data stored for later use
- Ext Entity/Terminator
	- square
	- ext entity receiving data at end of process
